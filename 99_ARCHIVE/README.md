# 99_ARCHIVE

## Purpose
Preserves the intellectual lineage of the framework. These are **historical** documents, not authoritative ones. They capture how the theory evolved, not what it currently claims.

## Important Distinction
- `01_CONCEPT_ARCHITECTURE/CURRENT_STATE.md` = what we currently believe
- `99_ARCHIVE/` = how we got here

Do NOT merge archived material directly into the active repository. Instead, extract from it using the classification-first protocol:

```
Source Material → Extraction → Classification → Repository
```
