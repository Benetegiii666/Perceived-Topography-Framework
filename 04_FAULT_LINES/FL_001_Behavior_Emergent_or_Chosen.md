# FL_001: Is Behavior Emergent or Chosen?

**Status:** Partially resolved (per seed material)
**Date:** 2026-06-09 (from seed material)

## The Tension

The core thesis claims behavior *emerges* from optimizer-topography interaction. But agents — especially human agents — experience behavior as *chosen*. These are different claims.

If behavior is emergent, agents are more like marbles on slopes. If behavior is chosen, agents have genuine agency that can override topographic gradients.

## Why It Matters

This is a foundational question. If the answer is "emergent," the framework has strong explanatory power but risks determinism. If the answer is "chosen," the framework needs to account for how agents override topographic pressure.

## Current Position

Partially resolved: behavior is *probabilistically* shaped by topography but not *deterministically* caused by it. Strong topographic gradients make certain behaviors highly probable; weak gradients leave more room for choice. This is a "both/and" rather than "either/or" — but the boundary conditions need work.
