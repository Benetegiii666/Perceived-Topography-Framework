# 04_FAULT_LINES

**Priority: 2ND HIGHEST — This is where breakthroughs will come from.**

Most researchers keep conclusions. Almost nobody systematically preserves contradictions. Future insights will emerge from unresolved tensions.

## Purpose
Contains contradictions, unresolved tensions, and conflicts between discoveries and the concept architecture. Fault lines are NOT bugs to fix — they are the pressure points where the theory is being tested.

## When to Create a Fault Line
- A discovery conflicts with the concept architecture
- Two aha moments point in opposite directions
- An emerging concept challenges an existing definition
- A counterargument exposes a genuine tension (not just disagreement)
