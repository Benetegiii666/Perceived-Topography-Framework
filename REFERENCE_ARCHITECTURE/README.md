# REFERENCE_ARCHITECTURE

## Purpose
High-Level Design and Reference Architecture for the Perceived Topography Framework. Translates the framework from design theory into implementable system design. Downstream of both the paper (conceptual authority) and the Build Discovery Guide (build-input authority).

## Design Stance
Design from reasoning-state preservation outward. Not from model choice, vendor choice, or generic RAG.
