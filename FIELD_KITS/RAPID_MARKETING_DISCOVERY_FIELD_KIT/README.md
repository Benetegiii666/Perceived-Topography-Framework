# Rapid Marketing Discovery Field Kit

## What This Is

This is Benet's practical working kit for conducting campaign brief workflow discovery with Rapid marketing stakeholders. It contains everything needed to orient stakeholders, conduct live discovery conversations, and capture answers that feed back into the formal repo workflow.

## What This Is Not

- Not the HLD, not the paper, not the build backlog, not an implementation artifact.
- Not authoritative over source artifacts. This folder contains working copies adapted for stakeholder conversations.
- Not Categories 4-13 discovery. This kit covers the first discovery pass only.

## Recommended Sequence

1. **Send the email** (`01_STAKEHOLDER_EMAIL.md`).
2. **Attach the executive orientation summary** (`02_EXECUTIVE_ORIENTATION_SUMMARY.md`) as pre-read.
3. **Open the meeting** with the live conversation script (`03_LIVE_CONVERSATION_SCRIPT.md`).
4. **Use the discovery questionnaire** (`04_DISCOVERY_QUESTIONNAIRE.md`) — choose Track A (recent campaign) or Track B (upcoming campaign).
5. **Capture notes** in the notes template (`05_NOTES_CAPTURE_TEMPLATE.md`).
6. **Map notes back to the repo** using the field-kit-to-repo mapping (`06_FIELD_KIT_TO_REPO_MAPPING.md`).

For deeper follow-up passes, use the full internal question bank (`04A_FULL_QUESTION_BANK_INTERNAL.md`).
